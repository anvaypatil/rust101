### Rust standard collections
- `Vec<T>`: Dynamic (growable) array
    - `vector` in C++
    - `ArrayList` in Java
    - `list` in Python
- `VecDeque<T>`: Double-ended queue
    - `deque` in C++
    - `ArrayDeque` in Java
    - `collections.deque` in Python
- `LinkedList<T>` : Doubly linked list
    - `list` in C++
    - `LinkedList` in Java
    - `-` in Python
- `BinaryHeap<T> where T : Ord`: Max Heap
    - `priority_queue` in C++
    - `PriorityQueue` in Java
    - `heapq` in Python
- `HashMap<K,V> where K: Eq+Hash`: Dictionary (Key-vale table)
    - `unordered_map` in C++
    - `HashMap` in Java
    - `dict` in Python
- `BTreeMap<K,V> where K: Ord`: Sorted dictionary
    - `map` in C++
    - `TreeMap` in Java
    - `-` in Python
- `HashSet<T> where T: Eq+Hash`: Hashtable or Set
    - `unordered_set` in C++
    - `HashSet` in Java
    - `set` in Python
- `BTreeSet<T> where T: Ord`: Sorted Set
    - `set` in C++
    - `TreeSet` in Java
    - `-` in Python
